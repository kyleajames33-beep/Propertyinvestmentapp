[ ] npm install @capacitor/core @capacitor/cli

SUPPORT
================================================================================

If you encounter issues:
1. Check the relevant guide file (listed above)
2. Review app-store-checklist.md troubleshooting section

4. Verify file paths and filenames match exactly

Common Issues:
- Icons not showing? → Check files are in /icons/ with exact names
- Screenshots wrong size? → Use Chrome DevTools device emulation
- PWA not installable? → Serve via HTTPS or localhost
- Build errors? → Check Node version, clear cache
