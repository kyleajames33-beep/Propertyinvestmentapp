# Execution Plan: Australian Property Investment Platform Build

## Overview
Multi-stage parallel agent swarm producing content, data, and technical spec for an NSW-focused property purchase guidance platform. Three personas: FHB-OO, INV-NEW, INV-EXP.

## Stage 1: Foundation (Parallel)
- **1A: Spec & Architecture Agent A** — Sitemap, user flows (3 personas), referral touchpoint mapping
- **1B: Spec & Architecture Agent B** — Data schemas, referral integration spec, v2/v3 roadmap
- **1C: Regulatory Research Agent** — NSW stamp duty, FHB schemes, land tax, APRA rules (all 4 sub-domains)

## Stage 2: Content Production (All Parallel, after Stage 1)
- **2A: Journey Content Writers** — 8 agents, one per stage, each producing 3 persona files
- **2B: Reference Library Writers** — 10 agents, one per reference doc
- **2C: Question Scripts Writer** — 1 agent, all 7 professional scripts
- **2D: Checklist Producer** — 1 agent, all stage-persona checklists
- **2E: Compliance & Disclaimers** — 1 agent, all compliance documents

## Stage 3: QA & Integration (After Stage 2)
- **3A: QA Agent** — Tone check, CTA verification, source verification, integration notes

## Output
Single /platform-build/ folder ready for developer handoff.
