# CTA Copy Library

## Version: 1.0 | Jurisdiction: NSW, Australia

This document provides three copy variants (soft, medium, direct) for every referral touchpoint defined in the platform's referral touchpoints map. All copy follows the platform's CTA principles: names the specific need, treats the user as the hirer, avoids false urgency, notes cost transparency where applicable, maintains contextual relevance, and acknowledges journey progression.

---

## General CTA Design Notes

### Button Text Guidelines

| Urgency Level | Button Text | Use Case |
|---------------|-------------|----------|
| Low (soft) | "Learn more", "Get started", "Find out how" | Early-stage users, soft touchpoints |
| Medium | "Connect now", "Get matched", "Book a consultation" | Users who have engaged with content |
| High (direct) | "Get your assessment", "Book today", "Start your application" | Users showing strong intent |

### Placement Rules

1. **Primary CTA** appears after the user has consumed the educational content for that stage. No form appears without context.
2. **Soft variant is default** -- displayed as the primary CTA text. Medium and direct variants shown via A/B testing or progressive disclosure (user clicks "learn more").
3. **Contextual anchoring** -- every CTA is tied to a specific learning moment on the page.
4. **No repeated referrals** -- if a user has already submitted a lead for a professional type, the CTA changes to status messaging.

### Urgency Levels

- **Soft**: No time pressure. Focus on education and exploration. User is still learning.
- **Medium**: Implies next step without pressure. Acknowledges the user's readiness.
- **Direct**: Clear call to immediate action. User has demonstrated intent through engagement.
